# WhatsApp Exports

Place WhatsApp exports here before ingesting them into the wiki.

Recommended naming:

- `raw/whatsapp/2026-05-09-family-group/_chat.txt`
- `raw/whatsapp/2026-05-09-friend-name/_chat.txt`
- `raw/whatsapp/2026-05-09-family-group-media/`

Keep these files immutable. If you export the same chat again later, create a new dated folder instead of replacing the old export.
