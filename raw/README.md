# Raw Sources

Store immutable source material here while importing it into the wiki. Do not edit raw files in place; add a new source or a dated replacement instead.

Raw content is part of this private project so the wiki has a complete source trail. Keep files local to the repository unless the group deliberately decides to publish or sync it elsewhere.

Suggested structure:

- `raw/whatsapp/` for exported WhatsApp chats and media.
- `raw/assets/` for downloaded images, documents, and other attachments.
- `raw/notes/` if you later add manually written source notes.
