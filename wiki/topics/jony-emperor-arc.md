# Jony Emperor Arc

> [!summary]
> Jony's long-term arc: become competent, build useful things, get rich, understand AI before the NPCs, and rule the galaxy with clean blue UI.

## Canon

[Jony](../people/jony-yadgar.md)'s long-term arc is becoming the first galactic emperor.

This is partly a joke, partly real ambition, and partly a worldview.

The emperor plan includes:

- become competent
- build useful things
- get rich eventually
- understand AI before the NPCs
- prepare for the robot future
- rule the galaxy with clean blue UI

## Why It Works As Lore

The joke is funny because Jony actually has the temperament for it: intense, practical, future-obsessed, and constantly trying to improve systems. The empire is absurd, but the ambition is not fake.
