# Exact Button Instructions Doctrine

> [!summary]
> Bad answer: vague theory. Good answer: click this, paste this, run this command.

## Canon

[Jony](../people/jony-yadgar.md) likes explanations that are simple, practical, exact, and copy-pasteable.

## Preferred Answer Style

- TLDR
- simple
- direct
- practical
- exact
- not corporate
- sometimes Hebrew
- sometimes English
- mixed when natural

## Common Requests

- "Make it shorter."
- "Make it more professional."
- "Don't change the meaning."
- "What do I click?"
- "Are you sure?"
- "Search online."
- "Just tell me what to do."

## Machine Submission Pattern

1. Machine does something stupid.
2. Jony sends screenshot.
3. Jony demands exact button instructions.
4. The machine eventually submits.
