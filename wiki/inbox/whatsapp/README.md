# WhatsApp Inbox

Converted WhatsApp transcripts go here before synthesis.

Generated transcripts in this folder are part of the wiki's source trail.

Run the local converter from the repository root:

```sh
python3 tools/whatsapp_export_to_markdown.py raw/whatsapp/2026-05-09-family-group/_chat.txt
```

Review the generated Markdown before using it to update durable wiki pages.
